# Redmine Diary

Redmine plugin to use time requests for reporting daily activity

## Install

Just copy it in the plugins directory and restart Redmine.

## Support

Tested with Redmine 2.3.1 and 2.3.2 and both Ruby 1.9.3 and 1.8.7.
